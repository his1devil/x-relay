为什么是这一对

终端在 Mac,掌控在手心。

🖥️

原生托管,不是镜像壳

vibTTY 基于 Ghostty 引擎,Claude/Codex/TIL 跑在真实 PTY 里——本地是完整 TUI,远程是结构化频道,同一进程无缝切换。

📱

Discord 式会话流

每个 session 一个频道:Markdown、代码块、diff、终端输出全部结构化渲染;正在执行的任务在列表实时跳动计时。

🔐

权限握在手里

Agent 要执行敏感操作?推送到手机,一键 Allow / Deny——离开工位也不失控。

🛰️

盲中继 · 端到端加密

自托管 relay 只见密文与房间号;AES-256-GCM 密钥仅存在于扫码那一刻的两端。

🧭

多 Mac 指挥台

家里、公司、机房——每台设备独立开关,会话按主机分组,消息自动路由到正确的那台。
